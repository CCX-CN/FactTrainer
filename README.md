# FactTrainer

FactTrainer is an experimental AI agent training platform that explores how an AI agent can learn from user-provided knowledge.

## Overview

Most AI assistants rely on pre-trained knowledge. FactTrainer explores a different approach: creating agents that start from zero knowledge and acquire domain-specific knowledge through a controllable training process.

## Features

- Create zero-knowledge AI agents
- Learn from user-provided materials
- Transform information into structured knowledge
- Train and evaluate agent performance
- Answer questions based on learned knowledge
- Refuse questions without relevant knowledge

## Demo Workflow

1. Create a new Agent
2. Provide learning materials
3. Analyze and structure knowledge
4. Train the Agent
5. Evaluate learning results
